### Core Lessons Learned
1. **Institutional Geometry:** "Stamps" (`rounded-md`) are more trusted than "Pills" (`rounded-full`) in government hospital contexts.
2. **Readability Floor:** `text-[10px]` is a high-severity accessibility failure in bright wards. Minimum floor is now `11px/12px` (text-xs).
3. **Tactile Certainty:** Subtle hovers fail under pressure. High-contrast hover/active states and subtle scale-down effects are mandatory for "Physical" UI feedback.
4. **Analytics Hierarchy:** Moving from a "List" to a "Control Center" with visual distribution charts significantly reduces cognitive load for staff.
